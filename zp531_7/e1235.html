ESSENCROE KLEVV UHS-I U3 A2 microSD Memory Card CRASシリーズ
UHSスピードクラス3 / ビデオスピードクラス V30 / アプリケーションパフォーマンスクラス2(A2)に対応した microSDXCカード 512GB

■4K動画も気軽に録画できるビデオスピードクラスV30に対応
30MB/secの書き込み最低保証速度持つビデオスピードクラスV30に対応しており、4K UHDの動画も安定して録画することができます。

■microSDXC / SDXC規格対応機器
※付属のSDカードアダプタを使用すればSDXC対応機器にも対応します。

■アプリケーションパフォーマンスクラス2(A2)対応
A2はランダムリード4000 IOPS、ランダムライト2000 IOPS、シーケンシャル10MB/s。A1規格に比べて、よりハイスピードなランダム性能を実現します。
※ IOPS（アイオプス） Input Output Per Second
アプリケーションの実行・保存で重視されるのは、ランダムの最低処理速度です。
最低低処理速度は、ランダムリード/ライト、シーケンシャルの2種類で表されます。
アプリケーションパフォーマンスクラス（Application Performance Class)は、アプリ実行やアプリのデータ保存に「最低必要な転送速度」を明示する
スマートフォンやタブレットのアプリ向けの規格です。
アプリケーションパフォーマンスクラス対応のmicroSDカードを使うとアプリの実行がスムーズに快適におこなえます。
※ご使用になられるスマートフォン、タブレット、ＰＣ等の本体がA2規格に対応している必要があります。
