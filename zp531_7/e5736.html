- Features -
Function 01
小鳥

つぶらな瞳にまるいからだ。おもわず目がほころんでしまう、とってもかわいい小鳥が3羽。微笑ましいピースフルなシーンをデザインに採用しました。
   
Function 02
ピースフル

素材に木を使うことで、優しい印象と温かみのある仕上がりに。振り子仕様を取り入れることで奥行きが生まれ、よりクラフトテイストを感じていただけます。
   
Function 03
ダンス

木の上で小鳥たちがまるでダンスをしているかのように動きます。ひとたび壁で動き出せば、明るいエッセンスを振りまきます。愛らしいその姿に癒されてください。
   
Function 04
振り子仕様

時計と振り子を一つの電池で動かす一体成型。揺れる振り子が壁面のアクセントに。
 
Function 05
サイズ感

デザインと実用性のバランスを考慮したサイズ感を実現。

- Brand information -
 
「人が過ごす場所と時間を心地よく」

インターフォルムは1979年に照明器具のメーカーとして創業して以来、時計やインテリア雑貨にもフィールドを広げ、みなさまの日々の暮らしのひとときを心地よくするお手伝いをしたいという想いを込めて、オリジナルデザインの商品をご提案してきました。

インターフォルムという社名には、「世界に通用するデザインを」という意味が込められています。
そんな理想を持ち、世界のトレンドと、日本の緻密なデザインワークをかけ合わせた商品づくりを続けています。
